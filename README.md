## 📋 What is axon?

axon is a tool to analyze and organize your code. It turns your code into a clear map. This map helps you see how pieces of your code connect. You do not need any programming knowledge to use axon. It works with many types of code including Python and TypeScript. axon runs on Windows and guides you step-by-step to get started.

## 💻 How axon Works

axon helps you understand code without writing any code yourself. It builds a map, called a knowledge graph, showing all parts of your code and how they relate.

- The tool reads your code files.
- It finds connections, like which functions call others.
- It shows this data in a clear and visual way.
- You can search and explore your code easily.

This works for many languages including Python and TypeScript. It also helps find unused or “dead” code, so you can clean your projects if needed.

## 🔧 Using axon for Code Analysis

Try these simple ways to use axon after opening it:

- **Open a Code Folder:** Use the “Open” menu to select a folder that contains your code.
- **Start Indexing:** Click the button to scan and build the code map.
- **Explore the Graph:** View the results in the visual interface. Click on nodes to see detailed code parts.
- **Search Function:** Use the search box to find specific functions, variables, or files.
- **Dead Code Detection:** Check for unused code with the built-in detection feature.
- **Export Data:** Save your analysis results for later or for sharing with others.

## 🛠️ Command Line Tool (Optional)

For users who prefer typing commands, axon includes a simple command-line interface (CLI). You do not need this if you just want to browse code visually.

To use the CLI:

1. Open Command Prompt on Windows.
2. Navigate to the folder where axon is installed.
3. Type `axon --help` to see available commands.
4. Use commands like `axon index` to scan your code without opening the visual app.

This feature is useful for developers or advanced users but not required for normal use.
